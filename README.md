# Gaggiuino Local Profiler – Home Assistant Add-on Repository

Dieses Repository enthält das **Gaggiuino Local Profiler** Add-on für Home Assistant.

## Installation

1. In Home Assistant: **Einstellungen → Add-ons → Add-on-Store → ⋮ → Repositories**
2. URL dieses Repositories hinzufügen:
   ```
   https://github.com/mxkissnr/gaggiuino-local-profiler
   ```
3. **Gaggiuino Local Profiler** im Store suchen und installieren

## Add-ons in diesem Repository

### Gaggiuino Local Profiler

Lokales Dashboard zur Visualisierung und Archivierung von Espresso-Shots der Gaggiuino-Maschine.

- Automatischer Sync vom Gaggiuino-Controller
- Interaktiver Shot-Profiler mit Vergleichsmodus
- Konfigurierbare URL und Sync-Intervall
- Persistente Datenspeicherung
